# 😊 Facial Recognition - Quick Start

## What Is Facial Recognition Login?

**Skip password entry and login with your face instead!** Optional privacy-friendly biometric authentication.

---

## ⚡ 30-Second Setup

### Step 1: Go to Settings
```
Settings → Privacy & Security → Facial Recognition
```

### Step 2: Click "Setup"
```
Modal opens with video stream
```

### Step 3: Capture Your Face
```
Position face in camera
Click "Capture Face"
Face descriptor saved!
```

### Step 4: Login With Face
```
Next time: Enter email on login page
"Sign In With Face" button appears
Click button → show face → auto-login
```

---

## 🔐 How It Works

1. **Setup:** Your face is analyzed, mathematical descriptor extracted (128 numbers)
2. **Storage:** Descriptor saved in database (no image stored, just math)
3. **Login:** Your face compared against saved descriptor using AI
4. **Match:** If faces match → instant login
5. **No Match:** Error message, try again or use password

2. **Find** "Facial Recognition Login" section

## 🔧 Disable Facial Recognition

**In Settings:**

```
1. Go to Settings → Privacy & Security
2. Click "Disable" button (replaces Setup after enabled)
3. Confirm you want to disable
4. Facial data deleted
5. You can re-enable anytime
```

## 💡 Tips

**For Best Results:**
- 🕯️ Good lighting (natural light recommended)
- 📱 Camera 8-12 inches from face
- 👀 Face forward, looking at camera
- 😊 Neutral expression (you can smile)
- ❌ Avoid sunglasses/hats during setup
- ❌ Large shadows on face

**During Login:**
- 🕯️ Similar lighting as setup
- 📱 Same distance from camera
- 👀 Look directly at camera
- 😐 Similar expression to setup

## ❓ FAQ

**Q: How secure is facial recognition?**  
A: Very! Only mathematical descriptor stored, not images. Each person has unique descriptor.

**Q: Can someone else use my face to login?**  
A: Unlikely. Descriptor requires exact facial match. Spoofing attempts detected.

**Q: What if I don't have a camera?**  
A: Use password login instead. Facial recognition option won't appear.

**Q: Can I use it on my phone?**  
A: Yes! Most phones have front camera. Works on all modern browsers.

**Q: What if my appearance changes?**  
A: Significant changes (beard, glasses) might not match. You can re-enable and recapture.

**Q: Is my face data shared with anyone?**  
A: No! All processing happens on your device. Data never leaves your app.

**Q: Can I have multiple faces registered?**  
A: Currently one face per account. Could update to support multiple angles.

**Q: What if facial recognition fails?**  
A: Use password login instead. Fallback always available.
